IMAGE attributes

  1. src: Enter the source of image
  2. alt: Alternative, enter kinda description for the image.
    (Incase it doesn't load, this text will show up with a picture icon)
  3. title:This text will be displayed when you hover over the image
  4. width & height: Recommended to enter both. Incase only 'width' entered then, height would be adjusted according to image's proportion
    But both should be specified, it tells computer this much space is fixed for image
  5. loading: only 2 attributes : 'eager' & 'lazy'
    Eager is by default so not required
    'Lazy' HELPS TO LOAD THE PAGE FASTER...it loads an image only when we are close to scrolling
