Block Elements

Are those which create a space around themselves.
for eg:

  • this <p> paragraph element
  • this <h1> even other heading heirachy
  • this <hr> horizontal line
  • <address> tag is also block. Below is an address block:
    F1xx1 xxx Society xxx city, xxx District xxx State - ******

Space around block elements can be styled using CSS.
